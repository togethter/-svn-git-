创建共享版本库:{
每一个开发者机器上都有一个服务器的数据库
共享版本库的集中情况
1)搭建服务器，在服务器端设置共享版本库(正规公司)
2)把共享版本库创建在某一个文件夹中
3)把共享版本创建在某一个优盘里
4)把共享版本库托管在第三方的网站上面
 
01 创建共享版本库
git --bare init
02 项目经理要对共享版本库做初始化操作
[1]项目经理线连接到共享版本库，把版本库下载到本地
git clone 路径
[2]项目经理完成忽略操作
1)创建忽略文件.gitignore
touch .gitignore
2)设置该文件的内容搜索gitHub gitingore找到objective-c
3)把该文件添加到暂缓区中
git add .gitignore
4)把暂缓区中的内容提交到本地的版本库
git commit -m "创建忽略文件"
5)把本地的版本库提交到共享版本库
git push
}


共享版本库项目初始化
1)通过XCode代码,改代码内部默认会自动做一次添加操作
2)通过终端来把当前的项目提交到本地版本库中
git commit -m "初始化项目"
